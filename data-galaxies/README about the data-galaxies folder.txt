
This folder should contain one file for each of the galaxies to be analysed. The name of the file must has the following structure

[Galaxy name]_rotmod.dat

Example: NGC2403_rotmod.dat

The “_rotmod.dat” is the same ending used by the SPARC data sample. The data can be downloaded from http://astroweb.cwru.edu/SPARC/Rotmod_LTG.zip
